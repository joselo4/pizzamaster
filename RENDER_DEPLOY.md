# Render (Static Site) - Solución pantalla blanca

Si ves `export default "/assets/index-....html"` significa que `dist/index.html` no es HTML.

## Configuración en Render
- **Build Command**: `bash render-build.sh`
- **Publish Directory**: `dist`

Luego haz: **Manual Deploy → Clear build cache & deploy** para borrar caché de build.
